The banking system allows customers to manage bank accounts
and perform financial transactions such as deposits and withdrawals.

## 3. Main Use Cases

- Create Bank Account

- Withdraw Money

## 4. System Analysis
The system is designed to separate responsibilities clearly.
Each component handles a specific task, ensuring maintainability
and scalability.
